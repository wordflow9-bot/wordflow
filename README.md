# WordFlow — минималистичный Telegram-бот для изучения иностранных слов

**WordFlow** — это удобный Telegram-бот для изучения и тренировки иностранных слов. Добавляйте слова вручную или распознавайте их с фотографий, тренируйтесь в двух режимах и отслеживайте прогресс своего обучения.

Проект использует локальное хранилище SQLite и интуитивный многоуровневый интерфейс с иерархической навигацией.

## ✨ Основные возможности

### 📚 Управление словарём

- **Добавление слов:**
  - Ручное добавление слова с автоматическим переводом
  - Выбор перевода из предложенного варианта или ввод собственного
  - Распознавание текста с фотографий (OCR) и добавление распознанных слов

- **Просмотр списка:**
  - Полный список ваших слов с уровнем освоения (%)
  - Удаление отдельных слов из словаря
  - Очистка всего словаря с подтверждением

- **Защита от дубликатов:**
  - Автоматическая проверка на наличие слова в словаре
  - Понятное уведомление при попытке добавить уже существующее слово

### 🎓 Тренировки

- **Два режима тренировки:**
  - **RU→EN:** угадайте английский перевод русского слова
  - **EN→RU:** угадайте русский перевод английского слова

- **Отслеживание прогресса:**
  - Автоматический подсчёт уровня освоения каждого слова
  - Статистика по всему словарю (средний уровень, количество попыток)

### 🔤 Дополнительные функции

- **Переводчик:** быстрый перевод любого слова с добавлением в словарь по желанию
- **Статистика:** просмотр общего прогресса обучения

## 🔧 Основные компоненты

### SQLiteWordRepository (`app/repositories/sqlite_repositories.py`)

Управление данными словаря пользователя:

- `add_word(user_id, word)` — добавить слово с проверкой дубликатов
- `get_all(user_id)` — получить все слова пользователя
- `get_word_id(user_id, word)` — проверить наличие слова
- `delete_word(id_)` — удалить слово по ID
- `clear_user_words(user_id)` — удалить все слова пользователя
- `adjust_mastery(id_, delta_correct, delta_total)` — обновить статистику

**Таблица БД:**
```sql
CREATE TABLE words(
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER,
    word_ru TEXT,
    word_en TEXT,
    total_cnt INTEGER,          -- Количество попыток
    correct_cnt INTEGER,        -- Количество правильных ответов
    UNIQUE (word_ru, word_en)
);
```

### Interaction (`app/core/interaction.py`)

Главный класс управления сценариями и состояниями бота:

**Управление меню:**
- `main_menu()` — главное меню
- `library_menu()` — управление словарём (уровень 2)
- `library_edit_menu()` — редактирование словаря (уровень 3)

**Управление словами:**
- `add_word()` — добавить слово вручную
- `button_delete_word()` — выбрать слово для удаления
- `list_words()` — показать список слов
- `clear_all_words_confirm()` — подтверждение очистки

**Тренировки:**
- `button_tr_choose_mode()` — выбор режима тренировки
- `ask_question()` — генерация вопроса
- `check_translation()` — проверка ответа

**Другие функции:**
- `button_translator()` — режим переводчика
- `button_photo_mode()` — режим распознавания фото
- `process_photo()` — обработка распознанных слов

### Trainer (`app/core/trainer_impl.py`)

Логика тренировок:

- `generate_question(user_id, mode)` — выбрать случайное слово и сформировать вопрос
- `check_answer(user_id, word)` — проверить ответ пользователя
- Обновление уровня освоения в БД

### TelegramGateway (`app/bot/utils.py`)

Обёртка над pyTelegramBotAPI:

- `send_message()` — отправить сообщение с кнопками
- `edit_message_text()` — редактировать сообщение (изменение кнопок под одним сообщением)
- `delete_button()` — удалить кнопки сообщения
- `_parse_buttons()` — парсинг кнопок (поддержка обоих форматов)

### Models (`app/core/models.py`)

Модели данных:

- `Word` — пара (русское слово, английское слово)
- `UserWord` — слово пользователя с метриками обучения
- `Session` — состояние пользователя
- `SessionType` — типы состояний (main_menu, database_add_word, train_ru_check_answer и др.)

### Handlers (`app/bot/handlers.py`)

Обработчики Telegram-событий:

- Команды: `/start`, `/menu`, `/list`, `/clear`, `/stats`
- Callback обработчик для кнопок
- Обработчик текстовых сообщений
- Обработчик фотографий

## 💾 Управление состояниями

Каждому пользователю соответствует сессия со статусом:

| SessionType | Описание |
|-------------|---------|
| `main_menu` | Главное меню |
| `library_menu` | Управление словарём (уровень 2) |
| `library_edit_menu` | Редактирование словаря (уровень 3) |
| `database_add_word` | Режим добавления слова |
| `database_add_word_choose_translation` | Выбор/ввод перевода |
| `database_delete_word` | Выбор слова для удаления |
| `confirm_clear_all` | Подтверждение очистки |
| `train_choose_mode` | Выбор режима тренировки |
| `train_ru_check_answer` | Тренировка RU→EN |
| `train_en_check_answer` | Тренировка EN→RU |
| `translator` | Режим переводчика |
| `photo_mode` | Режим распознавания фото |

## 📊 Статистика обучения

Каждое слово отслеживает:

- `total_cnt` — количество попыток перевода
- `correct_cnt` — количество правильных ответов
- `mastery_level()` — уровень освоения = (correct_cnt / 4 * 100) %

## 🚀 Установка и запуск

### Требования
- Python 3.8+
- pip

### Пошаговая установка

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/wordflow9-bot/wordflow.git
   cd wordflow
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Создайте файл `.env`:**
   ```bash
   cp .env.example .env
   ```

   Заполните необходимые переменные:
   ```env
   TELEGRAM_BOT_TOKEN=your_bot_token_here
   DATABASE_URL=sqlite:///./data/words.db
   GOOGLE_APPLICATION_CREDENTIALS=path/to/credentials.json
   ```

4. **Запустите бота:**
   ```bash
   python main.py
   ```
